# Netflix Content Analysis and Visualization

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset obtained from Kaggle. The objective is to understand the structure of Netflix's content library, identify trends and patterns, visualize key findings, and generate business insights based on the analysis.

## Dataset
Source: Netflix Movies and TV Shows Dataset (Kaggle)

The dataset contains information about Netflix content, including:
* Title
* Content Type (Movie/TV Show)
* Director
* Cast
* Country
* Date Added
* Release Year
* Rating
* Duration
* Genre

## Objectives
* Load and inspect the dataset
* Clean and preprocess the data
* Perform Exploratory Data Analysis (EDA)
* Create multiple visualizations
* Identify trends and patterns
* Generate business recommendations

## Technologies Used
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Data Cleaning
The following preprocessing steps were performed:
* Handled missing values
* Removed duplicate records
* Converted date fields to datetime format
* Created additional features such as Year Added
* Standardized categorical values

## Exploratory Data Analysis
The analysis focused on:
1. Movies vs TV Shows distribution
2. Top content-producing countries
3. Content ratings distribution
4. Netflix content growth over time
5. Release year trends

## Visualizations
The project includes:

* Bar Chart
* Line Chart
* Histogram
* Scatter Plot
* Pie Chart
* Heatmap

## Key Insights

* Movies dominate Netflix's content catalog.
* The United States contributes the highest number of titles.
* Netflix experienced significant content growth between 2016 and 2019.
* Mature audience content (TV-MA and TV-14) is the most common.
* Most available content has been released after the year 2000.

## Business Recommendations
1. Increase investment in high-performing TV series to improve retention.
2. Expand partnerships in emerging content markets.
3. Diversify content offerings across age groups.
4. Strengthen international original content production.
5. Continue prioritizing recent and trending content.
